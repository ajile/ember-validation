(private, inner) initValidation()

Method creates validators for every object's attribute from the validationScheme. Validators are gets from the registry by names. Default namespace for them is validator:, that means that they are should be placed in the validators/ project directory.

Validators creates and wrappes by an attribute mediator. It has the same
interface as they are. The mediators triggers events every time when they
called.
